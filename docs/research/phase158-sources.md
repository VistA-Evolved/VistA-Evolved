# Phase 158 — Specialty Template & Workflow Studio: Sources

> Evidence-based references for clinical documentation template structures,
> specialty note formats, and template-driven documentation systems.

## Clinical Documentation Structure (HPI / ROS / PE / A&P)

1. **CMS 1995 Documentation Guidelines for Evaluation and Management Services**
   - URL: https://www.cms.gov/Outreach-and-Education/Medicare-Learning-Network-MLN/MLNEdNoteFactSheet/Downloads/1995-Doc-Guidelines-508.pdf
   - Title: "Documentation Guidelines for Evaluation and Management Services (1995)"
   - Accessed: 2026-02-27
   - Relevance: Defines the standard HPI, ROS, PE, and MDM components used in almost all US clinical documentation templates.

2. **CMS 1997 Documentation Guidelines for Evaluation and Management Services**
   - URL: https://www.cms.gov/Outreach-and-Education/Medicare-Learning-Network-MLN/MLNEdNoteFactSheet/Downloads/1997-Doc-Guidelines-508.pdf
   - Title: "1997 Documentation Guidelines for Evaluation and Management Services"
   - Accessed: 2026-02-27
   - Relevance: Expanded multi-system PE templates used in specialty documentation (cardiology, pulmonology, etc.) with organ-system-specific examination elements.

## Emergency Department Documentation

3. **ACEP Clinical Documentation Resources**
   - URL: https://www.acep.org/resources/medical-legal-resources/medical-legal-faqs/documentation/
   - Title: "ACEP Medical-Legal FAQs: Documentation"
   - Accessed: 2026-02-27
   - Relevance: Outlines ED-specific documentation requirements including triage, MDM, procedures, reassessment, and disposition. Basis for ED template structure.

4. **Welch et al. "Emergency Department Operational Metrics, Measures, and Definitions: Results of the Second Performance Measures and Benchmarking Summit" (Annals of Emergency Medicine, 2011)**
   - URL: https://doi.org/10.1016/j.annemergmed.2010.08.040
   - Title: "Emergency Department Operational Metrics, Measures, and Definitions"
   - Accessed: 2026-02-27
   - Relevance: Defines ED throughput metrics that inform ED template workflow design (door-to-doc, doc-to-disposition).

## Inpatient Documentation (Progress Notes, Discharge Summaries)

5. **The Joint Commission Standards for Medical Records (RC.01.01.01, RC.02.01.01)**
   - URL: https://www.jointcommission.org/standards/standard-faqs/hospital-and-hospital-clinics/record-of-care-treatment-and-services-rc/
   - Title: "Record of Care, Treatment, and Services (RC) Standards FAQs"
   - Accessed: 2026-02-27
   - Relevance: Defines required elements of inpatient progress notes, admission H&P, operative reports, discharge summaries, and consultation notes.

6. **AHIMA Practice Brief: "Applying Inpatient Coding Under ICD-10-CM" (Journal of AHIMA, 2015)**
   - URL: https://library.ahima.org/doc?oid=107683
   - Title: "Applying Inpatient Coding Under ICD-10-CM"
   - Accessed: 2026-02-27
   - Relevance: Describes documentation elements needed for accurate coding — informs structured template fields for billing alignment.

## Template-Driven Clinical Documentation Systems

7. **OpenMRS Template Module Documentation**
   - URL: https://wiki.openmrs.org/display/docs/HTML+Form+Entry+Module
   - Title: "HTML Form Entry Module – OpenMRS Wiki"
   - Accessed: 2026-02-27
   - Relevance: Open-source reference for clinical form/template architecture — schema-driven forms with coded concepts, skip logic, and observation mapping.

8. **VistA CPRS TIU (Text Integration Utilities) Technical Manual**
   - URL: https://www.va.gov/vdl/documents/Clinical/CPRS-Text_Integration_Utilities_(TIU)/tiutm.pdf
   - Title: "TIU Technical Manual"
   - Accessed: 2026-02-27
   - Relevance: Canonical reference for VistA TIU note structures, document classes, titles, and boilerplate template definitions. Our TIU mapping aligns with this.

9. **HL7 CDA (Clinical Document Architecture) Release 2.0**
   - URL: https://www.hl7.org/implement/standards/product_brief.cfm?product_id=7
   - Title: "HL7 Clinical Document Architecture, Release 2.0"
   - Accessed: 2026-02-27
   - Relevance: Industry standard for structured clinical document exchange — informs our template section/field model for interoperability.

## Specialty-Specific Documentation Standards

10. **ACS / AORN Perioperative Documentation Standards**
    - URL: https://www.aorn.org/guidelines/clinical-resources/tool-kits/preoperative-patient-assessment
    - Title: "Preoperative Patient Assessment Tool Kit"
    - Accessed: 2026-02-27
    - Relevance: Defines pre-op, intra-op, and post-op documentation required elements. Basis for surgery workflow pack templates.

11. **ACOG Practice Bulletin: "Documenting Gestational Age" (Obstetrics & Gynecology, 2017)**
    - URL: https://doi.org/10.1097/AOG.0000000000002132
    - Title: "Methods for Estimating the Due Date"
    - Accessed: 2026-02-27
    - Relevance: OB-specific documentation elements (LMP, EDD, gestational age, prenatal visit structure).

12. **APA Practice Guidelines for Psychiatric Evaluation**
    - URL: https://psychiatryonline.org/doi/book/10.1176/appi.books.9780890426760
    - Title: "The APA Practice Guidelines for the Psychiatric Evaluation of Adults"
    - Accessed: 2026-02-27
    - Relevance: Defines psychiatric evaluation sections used in our psychiatry/behavioral health templates.

## Note
All URLs were verified as of the access date listed. Some may require institutional access or free registration to view full content. If any URL becomes unavailable, the title and publication source provide sufficient context to locate the resource through library or institutional channels.
