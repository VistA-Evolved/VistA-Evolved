# Architecture Decisions (ADRs)

## Source of truth

All decisions are recorded in Notion:
Company HQ → VistA Evolved HQ → Decision Log (ADRs)

## Why this folder exists

This folder is for lightweight “snapshot” copies of key accepted decisions for developers who prefer reading in-repo.

## Naming convention

`ADR-###-short-title.md`

Example:
`ADR-004-pilot-scope-mvp-modules.md`

## Minimum contents for an ADR snapshot

- Context
- Decision
- Alternatives considered
- Consequences
- Revisit if
