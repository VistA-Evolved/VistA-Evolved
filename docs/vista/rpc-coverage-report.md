## How to generate

```bash
cd apps/api
npx tsx src/tools/vista/buildRpcCoverageMatrix.ts --api http://127.0.0.1:3001
```

This will:
1. Fetch the live RPC catalog from `/vista/rpc-catalog`
2. Compare against the normalized Vivian index (`data/vista/vivian/rpc_index.json`)
3. Write coverage artifacts to `data/vista/vivian/`:
   - `rpc_present.json` -- RPCs found in both Vivian and live VistA
   - `rpc_missing_vs_vivian.json` -- Vivian RPCs absent from live VistA
   - `rpc_extra_vs_vivian.json` -- Live RPCs not in Vivian index
4. Overwrite this file with a full coverage report
