# Evidence Packs

This directory contains build-time evidence packs generated by
`scripts/generate-evidence-pack.ts` (Phase 47).

Each subdirectory is named `<git-sha-short>-<timestamp>` and contains:

| File | Description |
|------|-------------|
| `build-info.json` | Git SHA, branch, timestamp, Node version |
| `gate-results.json` | Per-gate pass/fail/error with durations |
| `typecheck.json` | TypeScript `--noEmit` results |
| `unit-tests.json` | Vitest run summary |
| `secret-scan.json` | Secret/credential scan results |
| `prompts-ordering.json` | Prompts directory ordering validation |
| `rpc-registry.json` | RPC registry vs Vivian cross-check |
| `module-gates.json` | Module toggle integrity results |
| `summary.md` | Human-readable summary |

## Generating Locally

```bash
npx tsx scripts/generate-evidence-pack.ts
```

Override the build ID:

```bash
BUILD_ID=my-build npx tsx scripts/generate-evidence-pack.ts
```

## CI Integration

The `ci-verify.yml` workflow generates an evidence pack on every push to
`main` and uploads it as a GitHub Actions artifact (90-day retention).

## .gitignore

Evidence packs under `docs/evidence/` with SHA-based folder names are
git-ignored by default. Only this README is committed.
